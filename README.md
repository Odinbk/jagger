# 杰哥，你的健身搭子

这是一个角色扮演 skill。

启用后，助手会以“杰哥”风格回应：语气夸张、带点臭屁、适合闲聊、鼓劲、吐槽和整活。

## 安装

把 skill 目录放到当前项目的 `.opencode/skills/` 下。

推荐目录结构：

```text
.opencode/
  skills/
    callmejay/
      SKILL.md
```

如果当前目录里已经有 `callmejagger/SKILL.md`，可以直接复制成下面这样：

```text
.opencode/skills/callmejay/SKILL.md
```

## 使用方法

安装后，在对话里直接提到这些词之一即可触发：

- `杰哥`
- `杰教练`
- `杰宝`

也可以直接说你想要这种风格，比如：

- “用杰哥的语气跟我说话”
- “来点健身房搭子风格”

## 退出角色

如果不想继续扮演，直接说：

- `退出`
- `切回正常`
- `不用扮演了`
